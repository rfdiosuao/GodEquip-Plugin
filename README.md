# 🌌 终焉奇点 (The Singularity) - GodEquip Plugin

> **“当数值失去了意义，法则便只是神明手中的玩物。”**

## 📖 简介 (Introduction)
**GodEquip** 是一款为 Luminode 设计的“概念神级”装备。它引入了一套名为**“终焉奇点”**的终极装备，赋予玩家凌驾于物理法则之上的力量（无限属性、飞行、无敌），但同时也伴随着严苛的“代价”与诅咒。

这不仅仅是一个  装备插件，更是一个带有 **资源管理** 与 **高风险高回报** 玩法的系统。

---

## ✨ 核心特性 (Features)

### 1. 概念级武装
*   **无限属性**：护甲、韧性、攻击力均为数学意义上的“无限” (Safe Infinity)。
*   **绝对防御**：只要持有者体内仍有能量（经验值 > 0），将免疫**所有**形式的伤害（包括 /kill、虚空、监守者音波）。
*   **维度飞行**：穿戴全套装备自动获得创造模式般的飞行能力。

### 2. 独特的视觉风格
*   **外观**：全套下界合金装备，自动应用 **寂静 (Silence)** 盔甲纹饰与 **钻石** 材质，呈现出深邃的幽蓝色调。
*   **沉浸式反馈**：全中文的 Action Bar 提示与交互反馈。

### 3. “代价”系统 (The Price)
为了平衡无限的力量，持有者必须支付昂贵的代价：
*   **能量虹吸**：每秒扣除 **1000 XP**。一旦经验耗尽，装备将强制脱落，飞行失效。
*   **高维傲慢**：无法拾取任何掉落物。

*   **使用门槛**：
    *   **剑**：需要 **10000 XP** 才能造成无限伤害，否则只有 1 点伤害。每次攻击消耗 **10,000 XP**。
    *   **弓**：需要 **10,000 XP** 才能拉开弓弦。每次射击消耗 **10,000 XP**。

## 🛠️ 安装与使用 (Usage)

### 安装
1. 将 `GodEquip-1.2-SNAPSHOT.jar` 放入服务器的 `plugins` 文件夹。
2. 重启服务器。

### 命令
*   `/godset` - 获取整套“终焉奇点”装备。

### 权限
*   `godequip.admin` - 允许使用 `/godset` 命令。
*   `godequip.admin.fly` - 拥有此权限的玩家在使用套装时，不会受到飞行状态检查的强制剥夺（用于管理员调试）。

### 兼容性
*   **核心**：Paper / Spigot 1.20+ (需支持 Armor Trim API)。
*   **插件冲突**：已内置 **PDC 智能识别**，完美兼容其他飞行插件（如 CMI, Essentials, AdvancedEnchantments），不会误伤非 OP 玩家的飞行权限。
